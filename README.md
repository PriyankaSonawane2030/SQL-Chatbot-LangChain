## 📌 Project Overview

This project demonstrates how Large Language Models (LLMs) can simplify database interactions by allowing users to ask questions in natural language instead of writing SQL queries manually.

The application converts user questions into SQL queries using LangChain and an LLM, executes those queries on a SQLite database, and returns both the generated SQL query and the final response in an easy-to-understand format.

This project was developed during my AI/ML internship as part of learning conversational AI, prompt engineering, LangChain, and Natural Language to SQL (NL2SQL) systems.
